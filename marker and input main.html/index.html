Markers and Inputs

This website helps to push markers on the Maps. Either enter the co-ordinates in the boxes or just click on the map to place a marker on the map. Only place two markers on the map. You can also playaround by dragging the marker in the map to see how the box values change.

If you place more than two markers on the map mistakenly, kindly reload the page to start again.
